Class Runner

Класс, собирающий список напоминалок и затем выполняющий связанные с ними действия в порядке возрастания дат соответствующих событий. При этом выполнение действий и некоторые другие действия делегируются классу-наследнику.

Использование:

  • сконструировать
  • добавить напоминалки с использованием метода add
  • вызвать метод run
