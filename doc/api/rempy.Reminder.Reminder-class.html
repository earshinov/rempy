Class Reminder

source code

object --+
         |
        Reminder

Базовый класс напоминалки. Напоминалка объединяет условие, согласно которому выбираются даты; действие, которое надо выполнить для каждой даты (обычно печать сообщения); количество дней для заблаговременного предупреждения о событии.

Замечания о терминологии:

  • дата называется подпадающей под напоминалку, если она удовлетворяет условию, заданному в этой напоминалке;
  • событием называется ситуация, когда заданная дата подпадает под определённую напоминалку:

    event = (date, reminder) | date `matches` reminder
