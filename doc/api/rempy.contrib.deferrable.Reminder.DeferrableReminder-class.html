fromString(dateCondition, doneDate=None, chainReminderFactory=<function fromParser at 0xa40b3e4>, chainParserFactory=<class 'rempy.StringParser.ReminderParser'>, *args, **kwargs)
Static Method

source code 

Альтернативный метод конструирования объекта класса DeferrableReminder. Позволяет задать всё одной строкой. Формат строки описан в документации парсера DeferrableParser.

Parameters:
  • dateCondition - строка для разбора
  • doneDate - Если не None и в строке dateCondition не задана дата последнего выполнения, в качестве такой даты будет использовано значение этого параметра. Параметр должен быть объектом класса datetime.date.
  • chainReminderFactory - Фабрика для создания оборачиваемого объекта, которой будут переданы
    • объект класса StringParser
    • объект класса DateCondition
    • дополнительные параметры args и kwargs

    Фактически, в качестве этого параметра можно использовать метод fromString из класса, объект которого хочется обернуть.

  • chainParserFactory - Фабрика для создания парсера, который можно передать в chainReminderFactory
  • args - дополнительные параметры, которые будут переданы в chainReminderFactory
  • kwargs - дополнительные параметры, которые будут переданы в chainReminderFactory
Returns:
объект класса DeferrableReminder

See Also: DeferrableParser
