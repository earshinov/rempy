Module Hierarchy

  • rempy: Пакет предоставляет классы и функции для задания напоминалок и поиска дат, на которые они выпадают.
    • rempy.Action: Содержит иерархию классов Action
    • rempy.DateCondition: Содержит иерархию классов DateCondition
    • rempy.Reminder: Содержит иерархию классов Reminder
    • rempy.Runner: Содержит иерархию классов Runner и функцию main
    • rempy.StringParser: Содержит иерархию классов StringParser.
    • rempy.contrib: Расширения базовой функциональности rempy
      • rempy.contrib.deferrable: Позволяет создавать напоминалки, отслеживающие дату последнего выполнения.
        • rempy.contrib.deferrable.DateCondition: Содержит класс DeferrableDateCondition
        • rempy.contrib.deferrable.Reminder: Содержит класс DeferrableReminder
        • rempy.contrib.deferrable.StringParser: Содержит класс DeferrableParser
        • rempy.contrib.deferrable.tests: При запуске из командной строки запускает все unit-тесты, объявленные в пакете rempy.contrib.deferred.
    • rempy.tests: При запуске из командной строки запускает все unit-тесты, объявленные в пакете rempy.
    • rempy.utils: Вспомогательные функции и классы
      • rempy.utils.algorithms: Функции и классы, реализующие различные алгоритмы
      • rempy.utils.dates: Функции и классы для работы с датами
      • rempy.utils.functional: Функции для программирования «в функциональном стиле»
      • rempy.utils.strings: Функции и классы для работы со строками
