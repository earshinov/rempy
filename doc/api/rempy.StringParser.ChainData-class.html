Instance Variable Details [hide private]

namedOptionHandlers

Словарь обработчиков длинных опций. Ключом должна быть строка названия опции; значением — функция-обработчик. Функция должна принимать два параметра:
  • токен, следующий за токеном, содержащим название опции, или None, если токены закончились;
  • Iterator по токенам, установленный на следующий токен.

Функция должна возвращать токен, следующий за последним обработанным. Если токены кончаются, должен быть возвращён None. Iterable, переданный через параметры, должен быть установлен на следующий токен.

optionHandlers

Обработчики коротких опций. Каждый элемент списка должен представлять из себя кортеж из двух элементов. Первый элемент кортежа должен быть строкой, задающей начало конструкции. Второй элемент кортежа должен быть функцией, принимающей единственным параметром токен.

unparsedRemainderHandler

Если не None, обработчик, который вызывается в случае, если в исходной строке остались неразобранные токены. Функция должна принимать параметры:
  • сконструированный к этому моменту объект класса DateCondition;
  • первый неразобранный токен;
  • Iterable по токенам, установленный на следующий токен;
  • неразобранная часть строки.

Функция должна возвращать объект класса DateCondition.

Пример обработчика: DateNamedOptionParser.
