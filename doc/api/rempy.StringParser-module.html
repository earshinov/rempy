Module StringParser

source code

Содержит иерархию классов StringParser. Полезен разработчикам классов напоминалок, в том числе разработчикам расширений. Конечным пользователям должен быть необходим редко.

При запуске из командной строки запускает присутствующие в модуле unit-тесты.
