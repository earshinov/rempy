wrapDate(unsafeDate, nonexistingDaysHandling=NonExistingDaysHandling.WRAP)

source code 

Преобразовать объект класса UnsafeDate в объект класса datetime.date.

Parameters:
  • unsafeDate - объект класса UnsafeDate
  • nonexistingDaysHandling - элемент «перечисления» NonExistingDaysHandling, задающий способ обработки ситуации, когда unsafeDate содержит несуществующую дату:
    • если WRAP, будет возвращён последний день месяца
    • если SKIP, будет возвращено значение None
    • если RAISE, будет выброшено исключение ValueError
Returns:
объект класса datetime.date или, если unsafeDate содержит несуществующую дату, в завимости от значения nonexistingDaysHandling

wrapDate_noFail(unsafeDate, nonexistingDaysHandling)

source code 

Преобразовать объект класса UnsafeDate в объект класса datetime.date.

Parameters:
  • unsafeDate - объект класса UnsafeDate
  • nonexistingDaysHandling - элемент «перечисления» NonExistingDaysHandling, задающий способ обработки ситуации, когда unsafeDate содержит несуществующую дату. В случае WRAP или SKIP первым элементов возвращаемого кортежа будет последний день месяца, вторым - False в случае WRAP, True в случае SKIP.
Returns:
кортеж из двух элементов
  • объект класса datetime.date
  • логическое значение, равное True, если эту дату следует «пропусить»
